      to ask questions using NLP question-answer using the provided knowledgebase document 
      (the current document example is based on the information from the API).
    </p>
  </div>
  <div class="section">
    <h2>Features</h2>
    <ul>
      <li>Fetching financial details for a particular company from the API</li>
      <li>Responding to natural language questions using the provided knowledgebase</li>
      <li>When using natural language, most relevant text is identified and retrieved</li>
      <li>Deployed via Telegram</li>
    </ul>
  </div>
